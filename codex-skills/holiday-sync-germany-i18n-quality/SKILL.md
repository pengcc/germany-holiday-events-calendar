# Holiday Sync Germany I18n Quality

Use this skill for locale routing, user-facing copy, date formatting, or translation changes.

## Rules

- Read the project guideline and `references/i18n.md`.
- Preserve explicit `/zh`, `/de`, and `/en` routes.
- Keep Chinese as the default entry point.
- Maintain user-facing copy in Chinese, German, and English.
- Keep development docs in English unless the user explicitly asks otherwise.
- Do not hard-code repeated user-facing copy where a locale dictionary is more maintainable.

## Verification

Check all locale routes affected by the change, including mobile text fit and date formatting.
