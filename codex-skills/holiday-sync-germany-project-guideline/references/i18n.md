# Internationalization

## Locale Routes

Use explicit static routes:

- `/zh`
- `/de`
- `/en`

Chinese is the default entry point.

## Copy Rules

- Product UI copy must be maintained in Chinese, German, and English.
- Development docs, plans, and skill files should be written in English.
- German holiday and state names should preserve official German names where useful.
- Do not hard-code user-facing strings inside reusable UI components when a locale dictionary is appropriate.

## Testing

For meaningful UI changes, verify:

- All three locale routes render.
- Key labels fit in desktop and mobile layouts.
- Date formatting is locale-appropriate.
- Shared links preserve explicit locale paths.
