# Holiday Sync Germany Frontend Quality

Use this skill for UI, layout, interaction, component, or styling work.

## Rules

- Read the project guideline and `references/ui-quality.md`.
- Use shadcn/ui, Tailwind CSS, Radix primitives, and lucide-react.
- Keep the visual direction clean, modern, and travel-planning oriented.
- Do not make the product feel like a generic admin dashboard.
- Do not rely on color alone to explain heatmap meaning.
- Ensure desktop and mobile text fit without overlap.
- Prefer custom calendar/heatmap UI for the main comparison view.

## Verification

For meaningful UI changes, run browser or Playwright checks across desktop and mobile once the app is runnable.
