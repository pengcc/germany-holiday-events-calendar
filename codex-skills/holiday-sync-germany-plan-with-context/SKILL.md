# Holiday Sync Germany Plan With Context

Use this skill when planning any Holiday Sync Germany task.

## Steps

1. Read `codex-skills/holiday-sync-germany-project-guideline/SKILL.md`.
2. Inspect the current repository state and branch.
3. Read the references relevant to the task.
4. Write a saved plan under `dev_locals/plans/` before execution.
5. Include scope, non-goals, files likely to change, checks to run, and cost/privacy/data assumptions.

## Plan Requirements

Every task gets a saved plan. Small tasks can use a short plan, but the plan still needs enough context for another session to understand what was intended.
